# web-exp

Песочница веб-экспериментов и UI-прототипов. Репозиторий нужен для быстрых standalone prototypes, например разных UX/UI вариантов dashboard.

## Как работать

- Один эксперимент должен иметь понятную папку или README-ссылку.
- Если прототип становится продуктовым, перенести его в repo-владелец.
- Не считать этот repo каноном продукта или production surface без отдельного решения.
